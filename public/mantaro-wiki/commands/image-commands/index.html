/danbooru

Returns an image from danbooru. You may add as many tags as you like. Each new word (separated by space) is considered a new Tag. Any additional tag. You may also entirely drop the rating when using multiple tags. Unless a different rating is specified, the Safe for Work Image rating will be used. Warning: This is a NSFW command. It will not work outside NSFW channels. Options rating Optional The image rating. Safe for Work Image is the default. tags Optional The image tag you're looking for. You can see a list of valid tags on the imageboard website. Ratings Safe for Work Image Questionable/Potentially Unsafe Image Not Safe for Work (NSFW/Explicit) Image Random (any of the above) Image Requiements This command can only be used in NSFW channels. Examples /danbooru /danbooru tags: yuri rating: Not Safe for Work (NSFW/Explicit) Image /danbooru rating: Random (any of the above) Image
