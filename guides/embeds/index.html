Introduction

Embeds are used to display rich content in messages. They are very powerful and can be used to display information in a variety of ways. In Mantaro, they are used to display information in the following ways:Welcome and leave messages Custom commands Birthday announcer In this page we will cover how to use embeds in Mantaro.
