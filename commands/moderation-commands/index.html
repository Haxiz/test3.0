"h3" ~>prune <amount> Description: Deletes messages (excluding messages that are older than 14 days). Minimum of messagesto delete is 5 and maximum is 100.Subcommand(s): bot {amount} : This will delete the specified amount of messages that were made by bots. nopins {amount} : This will delete the specified amount of messages that are not pinned. Requirements:This command requires https://github.com/Mantaro/MantaroBot/wiki/Command-reference-and-documentation#basic-knowledge Messages permissions.
